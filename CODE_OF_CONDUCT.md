# 貢献者行動規範（Contributor Covenant）

## 私たちの誓約

メンバー、コントリビューター、リーダーである私たちは、年齢・体格・目に見える/見えない障害・民族性・性的特徴・性自認と性表現・経験の程度・教育・社会経済的地位・国籍・容姿・人種・宗教・性的指向にかかわらず、コミュニティへの参加が誰にとってもハラスメントのない体験となるようにすることを誓約します。

## 私たちの基準

コミュニティにとって望ましい行動の例:

- 他者への共感と思いやりを示す
- 異なる意見・視点・経験を尊重する
- 建設的なフィードバックを行い、また快く受け入れる
- 自らの過ちの責任を認めて謝罪し、その経験から学ぶ
- 個人ではなくコミュニティ全体にとって最善となることに注力する

許容されない行動の例:

- 性的な言動や注目、あらゆる形態の性的なアプローチ
- 煽り、侮辱・中傷的なコメント、個人攻撃や政治的攻撃
- 公的・私的を問わないハラスメント
- 明示的な許可なく他者の私的情報（物理的・電子的な住所等）を公開すること
- 職業上不適切と合理的にみなされるその他の行為

## 執行責任

コミュニティのリーダーは、許容される行動の基準を明確化・執行する責任を負い、不適切・脅迫的・攻撃的・有害とみなした行動に対して、適切かつ公正な是正措置を講じます。

## 適用範囲

本行動規範は、コミュニティのすべての空間に適用され、また個人がコミュニティを公的に代表する場合にも適用されます。

## 執行

虐待・ハラスメント・その他の許容されない行為は、リポジトリのメンテナ（[@ohzono](https://github.com/ohzono)）まで GitHub を通じて報告してください。すべての申し立ては速やかかつ公正に検討・調査され、リーダーは報告者のプライバシーと安全を尊重します。

## 帰属
